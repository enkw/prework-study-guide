HTML

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A box-shadow adds shadow effects around an element's frame (multiple effects separated by commas can be added to adjust X and Y offsets relative to the element, blur and spread radius, and color)

Git

  • git status: Checks what branch is currently being worked on
  • git checkout -b branch-name: Creates a new branch and switches to it

CMD Line

  • pwd: Displays current directory (folder)
  • ls: Displays the list of files and/or folders in the current directory
  • cd: Navigates between directories in the path
  • cd ..: Navigates backwards, one level at a time, through the path of your directories
  • mkdir: Creates a new directory
  • touch: Creates a file inside a directory
